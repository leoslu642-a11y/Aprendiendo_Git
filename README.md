APRENDIENDO GIT-HUB
Objetivo: Crear una aplicación que muestre un curso para que oriente al usuario a hacer el proceso de subir código y a aprender los conceptos básicos de Git.

Actores del sistema: Usario aprendiz/principiante, Curso.
H.U.: 
Historia de Usuario 1: Iniciar el curso.
Historia de Usuario 2: Mostar cada uno de los contenidos del curso. 
Historia de Usuario 3: Ver secciones del curso una por una.
Hisotria de Usuario 4: Tener pausas entre secciones.
Hisotria de Usuario 5: Limpieza de pantalla si asi lo desea el usuario.
Historia de Usuario 6: Recibir retoralimentacion del cruso.
Historia de Usuario 7: Personalizar la experiencia del curso.
Historia de Usuario 8: Repetir el curso si asi lo pide el usuario.
